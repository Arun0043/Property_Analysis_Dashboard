Property Analysis Dashboard using Power BI

Project Objective:
Analyze property listings data to understand geographic distribution, pricing trends, property features, and key performance metrics.

Dashboard Pages:
1. State Analysis
2. Geographic Analysis
3. Property Feature Analysis
4. Price Comparison
5. KPI Dashboard

Tools Used:
- Power BI
- Excel
- DAX

Key Insights:
- Property listings are distributed across multiple states.
- Geographic clustering is visible in several regions.
- Property area and bedroom count influence pricing.
- Market estimates and rent estimates support valuation analysis.
